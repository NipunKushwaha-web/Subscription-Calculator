A sleek, responsive web application designed to help users calculate, track, and manage their recurring subscriptions in one place. Built with performance and usability in mind.

🔗 **Live Demo:** https://subscription-cal.netlify.app/

## ✨ Features

- **Expense Tracking:** Easily calculate your total monthly and yearly subscription costs.
- **Interactive UI:** A fast and responsive user interface powered by React.
- **Real-time Updates:** Instant calculation rendering as you add, edit, or remove subscriptions.
- **Cross-Device Compatibility:** Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices.

## 🛠️ Tech Stack

This project leverages modern web development tools for an optimized developer and user experience:

- **Frontend Framework:** React
- **Build Tool:** Vite (for Lightning-fast HMR and optimized builds)
- **Language:** JavaScript
- **Styling:** CSS3
- **Deployment:** Netlify
